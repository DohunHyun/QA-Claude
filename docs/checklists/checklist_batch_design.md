## 4. 배치Job명세 시트 검증

### [개선] 필수 컬럼 존재
- 단위업무 / 배치Job ID / 배치Job명 / 배치Job설명 / Step / 프로그램ID / 프로그램명 / 관련 DB 및 File / 작업성격 / 선행조건

### [개선] 배치Job ID-AN07 정합성
- 본 시트의 배치Job ID가 배치Job목록(AN07)에 동일하게 존재하는가? (정의서에 없는 ID 설계 시 개선 — §5.1 배치 정합성)

### [개선] Step 일련성
- Step이 1부터 누락 없이 순차 부여되어 있는가?

### [개선] 작업성격 enum 준수
- INSERT DB / UPDATE DB / SELECT DB / DELETE DB / UNLOAD DB / LOAD DB / SORT FILE / CREATE FILE / DELETE FILE 외 값 → 개선

### [개선] 프로그램ID 형식
- Java 기반: Batch Service Class명, Tool 기반(SORT/UNLOAD/LOAD): Shell 스크립트명이 기재되어 있는가? (공백이면 개선)

### [개선] 출력항목 정의 완전성

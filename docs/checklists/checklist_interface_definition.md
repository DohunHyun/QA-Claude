# 인터페이스정의서 체크리스트

**산출물명**: 인터페이스정의서
**샘플 문서번호**: NHXXX-단위업무코드-AN08
**기준**: NH 표준 템플릿 (v1.0) — 실제 양식 재검증 후 일부 enum 보정
**검증 단계**: 분석단계 (2차)

> ⚠️ **v1.0 대비 변경사항**: 연계방식 허용값에 "GSE", "대외MCA"를 추가하고 "SFTP"를 제거(실제 작성가이드 기준값: EAI/MFT/NAS/대외MCA/Socket/HTTP/GSE). 대외 인터페이스 판단 기준(작성가이드 주의사항) 검증항목 추가.

## 2. 표지 탭 검증

- 제·개정일자가 PM 등록 프로젝트 기간(착수일~종료일) 이내인가?

- 형식: NH[프로젝트코드]-[단위업무코드(2~3자리)]-AN08

- 연계방식="EAI" → 솔루션ID(EAI) 필수
- 개발구분="삭제" → 비고에 삭제사유 필수
